# Third-party components

## React Bits EvilEye

- Source: https://reactbits.dev/backgrounds/evil-eye (vendored from https://reactbits.dev/r/EvilEye-TS-CSS)
- Author: David Haz (React Bits, https://github.com/DavidHDev/react-bits)
- Licence: MIT + Commons Clause License Condition v1.0. The full licence text is
  vendored at `src/renderer/src/components/EvilEye/REACT-BITS-LICENSE.md`.
- The Commons Clause permits using the component inside an application such as
  this one, but forbids selling or redistributing the components themselves.

Local adaptations (documented in the component header): transparent shader
output, prop-driven desktop-wide gaze, uniform updates without WebGL
re-initialisation, and DPI-aware rendering.
